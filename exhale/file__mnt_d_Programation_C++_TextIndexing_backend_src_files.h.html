This header file defines the Struct unsigned_ints, and various template function which makes easier to read and write files using std:map-s or std::vector-s of std::pair-s. Struct unsigned_ints stores two uint16_t can be constructed/casted from/to a uint32_t value, interpreting uint32_t’s 16 first bits as the first value and the remaining bits as the second value.
