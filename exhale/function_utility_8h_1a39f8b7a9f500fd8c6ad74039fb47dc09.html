Template Function mapIntersection¶

  • Defined in File utility.h

Function Documentation¶

template<typename Key, typename Value, typename MergedValue, class Aggregator_t>
std::map<Key, MergedValue> mapIntersection(const std::map<Key, Value> &lhs, const std::map<Key, Value> &rhs, Aggregator_t aggregator)¶

Template function returning the intersection of two given std::map. Key and Value template parameters of boths std::map should be the same.

Return

std::map<Key,MergedValue> map intersection which associates common lhs and rhs keys with an aggregated value of type MergedValue

Template Parameters

  • KeyValue: Key and Value typenames stored by input std:map lhs and rhs

  • MergedValueAggregator_t: Aggregator_t typename must be a functor or function pointer type returning a MergedValue type (or a return type impliclty castable to MergedValue type).

Parameters

  • lhsrhs: Input maps to be merged

  • aggregator: Aggregation function pointer or functor wich takes Value values from lhs and rhs and returns an aggregated value of MergedValue type
